# OmniWorld AI Player Controller Design

## Purpose

Control the player character inside the world.

## Movement

Includes:

- Walking
- Running
- Jumping
- Camera movement

## Interaction

Player can:

- Talk with NPCs
- Use objects
- Build
- Access inventory

## Mobile Support

Includes:

- Virtual joystick
- Touch buttons
- Camera gestures

## Goal

Create smooth and responsive player controls.
