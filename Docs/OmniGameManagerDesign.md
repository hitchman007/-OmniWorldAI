# OmniWorld AI OmniGameManager Design

## Purpose

Control the main state of OmniWorld AI.

## Responsibilities

Manages:

- Game initialization
- System startup
- Global settings
- Game state

## Main Functions

Includes:

- InitializeGame()
- LoadSystems()
- StartWorld()
- SaveProgress()

## Connected Systems

Communicates with:

- Scene Manager
- Data Manager
- AI Core
- Player Systems

## Goal

Create the central controller of the OmniWorld AI experience.
