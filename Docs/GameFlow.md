# OmniWorld AI Game Flow

## Start

Launch OmniWorld AI

## Main Menu

Player selects:

- Start Journey
- Create Avatar
- Create World
- OmniGuide

## Avatar Creation

Player creates identity.

## NeoGenesis Hub

Player enters first world.

## Exploration

Player can:

- Move
- Interact
- Build
- Complete quests

## Save

Player progress is stored.
