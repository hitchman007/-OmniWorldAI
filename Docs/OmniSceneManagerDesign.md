# OmniWorld AI OmniSceneManager Design

## Purpose

Manage scene loading and transitions.

## Responsibilities

Controls:

- Main menu loading
- World loading
- Scene transitions
- Scene states

## Main Functions

Includes:

- LoadScene()
- ChangeScene()
- UnloadScene()
- RestartScene()

## Connected Systems

Communicates with:

- Game Manager
- World Manager
- Player System
- Save System

## Goal

Create smooth and organized scene management.
