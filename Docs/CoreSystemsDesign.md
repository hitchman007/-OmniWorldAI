# OmniWorld AI Core Systems Design

## Purpose

Define the main systems that control the application.

## Game Manager

Controls:

- Game initialization
- System startup
- Global states

## Scene Manager

Controls:

- Scene loading
- Scene transitions
- World access

## Data Manager

Controls:

- Player data
- World data
- Save systems

## Communication

Systems communicate through:

- Events
- Shared data
- Service connections

## Goal

Create a strong foundation for OmniWorld AI.
