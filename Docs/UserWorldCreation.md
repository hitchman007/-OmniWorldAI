# OmniWorld AI User World Creation

## World Creator

Users can:

- Create worlds
- Customize environments
- Place objects

## Creation Tools

Includes:

- Terrain editor
- Building tools
- Object placement
- Environment settings

## Saving

- World ID
- Creator ID
- World data

## Sharing

Users can:

- Publish worlds
- Invite players
- Create communities

Enable users to become creators inside OmniWorld AI.
