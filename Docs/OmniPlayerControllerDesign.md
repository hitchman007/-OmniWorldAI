# OmniWorld AI OmniPlayerController Design

## Purpose

Control player movement and interaction.

## Responsibilities

Manages:

- Movement
- Camera control
- Player actions
- Interaction

## Movement Systems

Includes:

- Walk
- Run
- Jump
- Mobile controls

## Interaction

Handles:

- Object interaction
- NPC interaction
- Building actions
- Inventory access

## Connected Systems

Communicates with:

- Avatar System
- Inventory System
- World System
- UI System

## Goal

Create a responsive and immersive player controller.
