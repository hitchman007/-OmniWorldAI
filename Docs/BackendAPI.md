# OmniWorld AI Backend API

## Purpose

Connect game systems with online services.

## Services

### Account Service

Handles:

- Login
- Profiles
- Security

### World Service

Handles:

- World loading
- Saving
- Sharing

### Player Service

Handles:

- Progress
- Inventory
- Achievements

### AI Service

Handles:

- AI requests
- NPC data
- Generated content

## Goal

Create reliable communication between OmniWorld AI clients and servers.
